# Cierre Mexicano — Public Releases

This repository hosts:

- **`manifest.json`** — auto-update manifest consumed by Cierre Mexicano
  clients at startup
- **GitHub Releases** (`v*` tags) — public download URLs for the Windows
  installer `CierreMexicano-Setup.exe`

The application source code is in a separate private repository.

## For users

You don't need to interact with this repository directly. The Cierre
Mexicano application checks for updates automatically. To manually check,
open the app and click **"Buscar actualizaciones"** in the lower-right
corner.

## For the maintainer

See `RELEASE.md` in the private source repository for the publishing flow.

## Support

For support, contact the maintainer through your usual channel.
